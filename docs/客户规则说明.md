# 利润与主播工资规则说明（飞书规则表镜像）

> 唯一生效口径以飞书多维表格 `15_工资规则说明` 为准。本文件只用于部署人员离线查看，不新增规则。

1. 人员、所属店铺、基本工资和提成百分比，以飞书 `13_人员表` 为准；客户只维护这些参数，不修改系统公式。
2. 同一店铺可以对应多名员工，每名员工按自己的基本工资和提成比例独立计算。
3. 主播提成基数默认采用快递助手 ERP 毛利报表返回的店铺最终净利润（`netSalesProfit`）。ERP 报表已经综合退款、退货成本、实际货品成本及运费，因此工资表不再二次扣减，避免重复扣款。
4. 店铺利润小于 0 时，提成按 0 计算，不产生负提成。
5. 工资公式：`基本工资 + 绩效工资 + 奖金 + MAX(0, 店铺净利润) × 提成百分比 - 扣款`。
6. 绩效工资、奖金、扣款由客户在每月工资记录中填写；未填写按 0 计算。
7. 每月 15 日结算上月工资，统一保留小数点后两位。
8. 月结时系统保存店铺利润和提成快照，并将工资标记为“已结算”。已结算工资不跟随之后的数据刷新自动变化。
9. 15 日之后才确认的历史异常售后，不直接改动已结算工资；如客户确认需要追溯，在下月作为奖金或扣款调整并保留说明。
